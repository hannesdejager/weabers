This is just a sample Web application. To edit this template you can edit the 'index.html' file found under XXX\apps where XXX is the installation directory that you've chosen (Normally "c:\Program Files\Inspired Archetypes\Weabers")

You can add more than one web application to be executed by Weabers. Play around, it works exactly like web programming.

Also download and install PHP. Weabers can make use of PHP or other CGI scripting methods. To enable PHP:

  1. Download and Install PHP
  2. Edit the file onInit.lua under the weabers installation directory
  3. Add the line SetPHPPath("C:\\PHP\\php-cgi.exe"); to the end of the file. (Note replace C:\\PHP with your PHP directory.)
  4. Link to some PHP files and run Weabers.

You can also:

  • modify the main menu items by editing onBuildMainMenu.lua.
  • modify the application title/caption, php path and app path by editing onInit.lua.
  • You can specify the initial url to open when Weabers start by editing onReady.lua.
